Power8 支持

  • vaneCloud作为一款基于openstack的云计算平台产品，对于计算平台的支持也相当丰富。vaneCloud默认使用X86 KVM虚拟化技术作为默认计算平台，同时能够兼容VMWare虚拟化技术，甚至于多用于金融业的IBM Power计算平台。
  • 对于IBM Power计算平台，vaneCloud提供了2种支持方式，可以满足大多数使用IBM Power服务器的用户的需要。

PowerVM 虚拟化接入

Glance
镜像服务，并与PowerVC的Glance服务交互，提供镜像同步、操作等功能
Storage Drivers
PowerVC存储驱动，含常见IBM存储、EMC存储及存储交换机驱动
PowerVM Drivers
IBM PowerVM的OpenStack nova服务驱动
Network Drivers
IBM PowerVM的OpenStack neutron服务驱动
AMQP
队列服务，并与PowerVC的AMQP服务交互，提供消息队列的直接对接
DBMS
vaneCloud数据库，存储vaneCloud平台数据
Cinder
块存储服务，并与PowerVC的Cinder服务交互，提供块存储及存储网络信息的同步、操作等功能
Nova
计算服务，并与PowerVC的Nova服务交互，提供计算服务信息的同步、操作等功能
Neutron
网络服务，并与PowerVC的Neutron服务交互，提供网络信息同步、操作等功能
PowerVC API
OpenStack 标准API
API additions
PowerVC提供的补充API
OVF
提供OVF打包格式
Flavors
计算模板服务
Quotas
配额服务
VM Mgmt
虚拟机管理服务
Keystone
vaneCloud认证服务，负责服务认证、用户认证
Platform
IBM EGO调度器
Scheduler
OpenStack标准调度器
Projects
Monitoring
vaneCloud监控服务，提供平台监控数据收集、存储功能
