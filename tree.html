Power8 支持

  • vaneCloud作为一款基于openstack的云计算平台产品，对于计算平台的支持也相当丰富。vaneCloud默认使用X86 KVM虚拟化技术作为默认计算平台，同时能够兼容VMWare虚拟化技术，甚至于多用于金融业的IBM Power计算平台。
  • 对于IBM Power计算平台，vaneCloud提供了2种支持方式，可以满足大多数使用IBM Power服务器的用户的需要。

PowerVM 虚拟化接入

Glance

PowerVM虚拟化接入：镜像服务，并与PowerVC的Glance服务交互，提供镜像同步、操作等功能

PowerKVM虚拟化接入：Glance镜像服务

Storage Drivers

PowerVM虚拟化接入：PowerVC存储驱动，含常见IBM存储、EMC存储及存储交换机驱动

PowerKVM虚拟化接入：vaneCloud存储驱动，含常见存储交换机驱动

PowerVM Drivers

PowerVM虚拟化接入：IBM PowerVM的OpenStack nova服务驱动

PowerKVM虚拟化接入：IBM PowerVM的OpenStack nova服务驱动

Network Drivers

PowerVM虚拟化接入：IBM PowerVM的OpenStack neutron服务驱动

PowerKVM虚拟化接入：PowerKVM neutron服务驱动

AMQP

PowerVM虚拟化接入：队列服务，并与PowerVC的AMQP服务交互，提供消息队列的直接对接

PowerKVM虚拟化接入：消息队列服务

DBMS

PowerVM虚拟化接入：vaneCloud数据库，存储vaneCloud平台数据

PowerKVM虚拟化接入：数据库服务，存储PowerVC信息存储

Cinder

PowerVM虚拟化接入：块存储服务，并与PowerVC的Cinder服务交互，提供块存储及存储网络信息的同步、操作等功能

PowerKVM虚拟化接入：OpenStack块存储服务

Nova

PowerVM虚拟化接入：计算服务，并与PowerVC的Nova服务交互，提供计算服务信息的同步、操作等功能

PowerKVM虚拟化接入：OpenStack计算服务

Neutron

PowerVM虚拟化接入：网络服务，并与PowerVC的Neutron服务交互，提供网络信息同步、操作等功能

PowerKVM虚拟化接入： OpenStack网络服务

PowerVC API

PowerVM虚拟化接入：OpenStack 标准API

PowerKVM虚拟化接入：OpenStack 标准API

API additions

PowerVM虚拟化接入：PowerVC提供的补充API

PowerKVM虚拟化接入：PowerVC提供的补充API

OVF

PowerVM虚拟化接入：提供OVF打包格式

PowerKVM虚拟化接入：提供OVF打包格式

Flavors

PowerVM虚拟化接入：计算模板服务

PowerKVM虚拟化接入：计算模板服务

Quotas

PowerVM虚拟化接入：配额服务

PowerKVM虚拟化接入：配额服务

VM Mgmt

PowerVM虚拟化接入：虚拟机管理服务

PowerKVM虚拟化接入：虚拟机管理服务

Keystone

PowerVM虚拟化接入：vaneCloud认证服务，负责服务认证、用户认证

PowerKVM虚拟化接入：认证服务

Platform

PowerVM虚拟化接入：IBM EGO调度器

PowerKVM虚拟化接入：IBM EGO调度器

Scheduler

PowerVM虚拟化接入：OpenStack标准调度器

PowerKVM虚拟化接入：OpenStack标准调度器

Monitoring

PowerVM虚拟化接入：vaneCloud监控服务，提供平台监控数据收集、存储功能

PowerKVM虚拟化接入：vaneCloud监控模块
