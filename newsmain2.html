2016 红帽峰会盛大召开 全面聚焦云计算和实践应用
2016/06/20
物联网（IOT）的出现，标志着数据世界正在迈向一个新的进程，物理与数据的界限正在被涂改，随之而来的新机遇与新挑战，让各行各业的领导者驻足、观望。物联网（IOT）的出现，标志着数据世界正在迈向一个新的进程，物理与数据的界限正在被涂改，随之而来的新机遇与新挑战，近日，中国第一家OpenSatck公司的SDN方案提供商BigSwitch、全球领先的IT服务商戴尔联合推出一款全新的云网络解决方案。 同一时期，UnitedStack有宣布与Big Switch Networks达成战略合作伙伴关系，双方将通过在SDN与OpenStack解决方案的深度合作，为全球企业及服务提供商的数据中心提供更便捷统一的云基础设施。 经过13个版本的迭代，OpenStack已经日渐成熟，成为开源云平台的事实标准。但是，在局部架构上仍然存在一些待优化的点，例如OpenStack网络解决方案在当前版本中仍无法实现对于包括物理交换机及虚拟交换机、虚拟网络在内的网络设备的统一部署和管理，需要登录每个设备进行单独配置，这从很大程度上影响了OpenStack的扩展性和可管理型。 为了帮助用户有效解决这一挑战，UnitedStack有云联合全球领先的SDN解决方案提供商Big Switch，基于Dell公司的网络设备，合作部署并联合测试了Big SwtichBig Cloud Fabric（BCF）方案，在OpenSatck环境下为用户提供可以有机结合白牌以太网交换设备和SDN控制器软件并扩展OpenStack的Horizon GUI简化租户工作流的网络设备统一部署和管理解决方案。 该解决方案包含： UnitedStack有云的操作系统UOS 3.0——包含标准化的OpenStack服务、全新的开发平台和新版本部署套件组成的先进DevOps体系，以及与之配套的文档服务体系。 Dell Networking交换机和服务器——包含一系列10/40 GbE开放式网络交换机及1RU机架式服务器 Big Cloud Fabric——支持通过Neutron Plugin无缝集成OpenStack，为生产级OpenStack部署提供所需的规模和弹性。016年5月10日，北京］近日，中国第一家OpenSatck公司Unitedstack有云与全球领先的SDN方案提供商BigSwitch、全球领先的IT服务商戴尔联合推出一款全新的云网络解决方案。 同一时期，UnitedStack有宣布与Big Switch Networks达成战略合作伙伴关系，双方将通过在SDN与OpenStack解决方案的深度合作，为全球企业及服务提供商的数据中心提供更便捷统一的云基础设施。 经过13个版本的迭代，OpenStack已经日渐成熟，成为开源云平台的事实标准。但是，在局部架构上仍然存在一些待优化的点，例如OpenStack网络解决方案在当前版本中仍无法实现对于包括物理交换机及虚拟交换机、虚拟网络在内的网络设备的统一部署和管理，需要登录每个设备进行单独配置，这从很大程度上影响了OpenStack的扩展性和可管理型。 为了帮助用户有效解决这一挑战，UnitedStack有云联合全球领先的SDN解决方案提供商Big Switch，基于Dell公司的网络设备，合作部署并联合测试了Big SwtichBig Cloud Fabric（BCF）方案，在OpenSatck环境下为用户提供可以有机结合白牌以太网交换设备和SDN控制器软件并扩展OpenStack的Horizon GUI简化租户工作流的网络设备统一部署和管理解决方案。 该解决方案包含： UnitedStack有云的操作系统UOS 3.0——包含标准化的OpenStack服务、全新的开发平台和新版本部署套件组成的先进DevOps体系，以及与之配套的文档服务体系。 Dell Networking交换机和服务器——包含一系列10/40 GbE开放式网络交换机及1RU机架式服务器 Big Cloud Fabric——支持通过Neutron Plugin无缝集成OpenStack，为生产级OpenStack部署提供所需的规模和弹性。
